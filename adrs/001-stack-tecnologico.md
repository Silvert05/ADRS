# ADR-001: Stack tecnológico para frontend y backend


## Contexto
Necesitamos un sistema web moderno, rápido, fácil de mantener y accesible para usuarios sin experiencia tecnológica.


## Decisión
- Frontend: React.js
- Backend: Node.js (mock API inicialmente)


## Opciones consideradas
1. Angular + Express → demasiado complejo para prototipo académico
2. Vue.js + Node.js → viable, pero React tiene mejor documentación y soporte


## Consecuencias
Proyecto modular, fácil de expandir y compatible con futuras integraciones
